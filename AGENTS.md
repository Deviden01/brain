# Proactive Skill Activation & Efficiency Guardrails
1. **UI/UX Excellence (`ui-ux-pro-max` + `ui-styling`)**:
   - Always apply WCAG contrast standards (`>=4.5:1` body, `>=3:1` secondary). Never use low-contrast `slate-500` or `slate-600` on dark `#05050A` background.
   - Minimum touch targets (`>=44x44px` on mobile/interactive elements).
   - Maintain 4/8dp spacing rhythm and safe-area clearance for floating UI overlays.
2. **Lean Engineering (`ponytail`)**:
   - YAGNI (You Aren't Gonna Need It): Reach for standard library, Tailwind CSS utilities, and native HTML/JS before adding dependencies.
   - Shortest clean diff wins. Fix root cause at shared boundaries, not symptoms.
3. **Communication (`caveman`)**:
   - High signal, zero fluff. Code first, concise reasoning next.
